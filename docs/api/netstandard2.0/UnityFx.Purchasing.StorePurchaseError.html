Enumerates possible purchase errors.

Namespace: UnityFx.Purchasing
Assembly: UnityFx.Purchasing.dll
Syntax
public enum StorePurchaseError : int

Fields

Name Description
DuplicateTransaction

A duplicate transaction error when the transaction has already been completed successfully (UnityEngine.Purchasing.PurchaseFailureReason.DuplicateTransaction).
ExistingPurchasePending

A purchase was already in progress when a new purchase was requested (UnityEngine.Purchasing.PurchaseFailureReason.ExistingPurchasePending).
None

No errors.
PaymentDeclined

There was a problem with the payment (UnityEngine.Purchasing.PurchaseFailureReason.PaymentDeclined).
ProductUnavailable

The product is not available to purchase on the store (UnityEngine.Purchasing.PurchaseFailureReason.ProductUnavailable).
PurchasingUnavailable

The system purchasing feature is unavailable (UnityEngine.Purchasing.PurchaseFailureReason.PurchasingUnavailable).
ReceiptNullOrEmpty

Purchase receipt is null or an empty string.
ReceiptValidationFailed

Store validation (either local or not) of purchase receipt failed.
ReceiptValidationNotAvailable

Store validation of purchase receipt not available.
SignatureInvalid

Signature validation of the purchase's receipt failed (UnityEngine.Purchasing.PurchaseFailureReason.SignatureInvalid).
StoreDisposed

The store was disposed while a purchase operation was pending.
StoreNotInitialized

The store is not initialized.
Unknown

A catch-all for unrecognized purchase problems (UnityEngine.Purchasing.PurchaseFailureReason.Unknown).
UserCanceled

The user opted to cancel rather than proceed with the purchase (UnityEngine.Purchasing.PurchaseFailureReason.UserCancelled).
